# Sample Character Placeholder

This file is a placeholder. Add your character images here.

## Getting Started

1. Download or create character images
2. Place them in this directory
3. Supported formats: PNG, JPG, JPEG
4. Recommended size: 1024x1024 pixels

## Example Images

You can use:
- Photos of yourself or others
- Anime/manga character artwork
- AI-generated characters
- VTuber models
- 3D rendered characters

See [docs/CHARACTER_GUIDE.md](../../docs/CHARACTER_GUIDE.md) for detailed instructions.

## Testing

Test your images:
```bash
python tools/test_character.py assets/characters/your_image.png
```
